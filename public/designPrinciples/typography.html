Emphasis Techniques:

The following emphasis techniques will be used.

Add emphasis by adding italics or obliques.

Add emphasis by using bold to highlight words.

Add EMPHASIS by using ALL CAPS to highlight words.

Add emphasis by changing the type size of words.

Add emphasis by using underlines to highlight links.
